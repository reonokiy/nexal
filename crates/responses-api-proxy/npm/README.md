# @openai/nexal-responses-api-proxy

<p align="center"><code>npm i -g @openai/nexal-responses-api-proxy</code> to install <code>nexal-responses-api-proxy</code></p>

This package distributes the prebuilt [Nexal Responses API proxy binary](https://github.com/openai/codex/tree/main/nexal-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/nexal-responses-api-proxy.js --help
```

Refer to [`nexal-rs/responses-api-proxy/README.md`](https://github.com/openai/codex/blob/main/nexal-rs/responses-api-proxy/README.md) for detailed documentation.
